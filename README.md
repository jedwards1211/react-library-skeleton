This is my personal skeleton for creating a React library npm package.  You are welcome to use it.

## Tools used

* babel 6
* mocha
* chai
* istanbul
* nyc
* babel-plugin-istanbul
* eslint
* eslint-watch
* flow
* flow-watch
* enzyme
* jsdom
* semantic-release
* husky
  * commitmsg: uses validate-commit-msg
  * precommit: runs lint and flow
  * prepush: runs test
* Travis CI
* Coveralls
